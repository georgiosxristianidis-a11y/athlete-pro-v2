# Plan 01: Server Refactor — Split server.js into routes/ + lib/

## Objective
Split the monolithic `server.js` (169 lines) into a thin entry point (~20 lines) plus dedicated route modules (`routes/coach.js`, `routes/integrations.js`) and a shared Anthropic client singleton (`lib/anthropicClient.js`). This eliminates the single-file server monolith and establishes a maintainable backend structure. No client-side code is touched, so regression risk is zero.

## Tasks

### Task 01-01-01: Create lib/anthropicClient.js

<read_first>
- server.js — see the current Anthropic SDK initialization on line 12 (`require('@anthropic-ai/sdk')`) and line 15 (`new Anthropic(...)`)
</read_first>

<action>
1. Create the directory `lib/` at the project root.
2. Create the file `lib/anthropicClient.js` with the following exact content:

```js
'use strict';
const Anthropic = require('@anthropic-ai/sdk');
const anthropic = new Anthropic({ apiKey: process.env.ANTHROPIC_API_KEY });
module.exports = anthropic;
```

This file exports a singleton Anthropic client instance initialized from the environment variable.
</action>

### Task 01-01-04: Rewrite server.js as thin entry point

<read_first>
- server.js — current full file (169 lines)
- routes/coach.js — the coach route module just created
- routes/integrations.js — the integrations route module just created
</read_first>

<action>
Replace the entire contents of `server.js` with:

```js
'use strict';
require('dotenv').config();
const express = require('express');
const path = require('path');

const app = express();
app.use(express.json());
app.use(express.static(path.join(__dirname)));

app.use('/api', require('./routes/coach'));
app.use('/api', require('./routes/integrations'));

function startServer(port = process.env.PORT || 3000) {
  return app.listen(port, () => {
    console.log(`\n  Athlete Pro  →  http://localhost:${port}\n`);
  });
}

module.exports = { app, startServer };

if (require.main === module) {
  startServer();
}
```

This file must be at most 25 lines (excluding blank lines in the template above). It must NOT contain:
- `new Anthropic`
- `app.post` or `app.get` (route registration is via `app.use`)
- `_buildSystemPrompt`
</action>
