**1. Add breakpoint custom properties to css/base.css :root block:**
```css
/* Breakpoints (reference values — used in @media, documented here) */
--bp-sm: 360px;
--bp-md: 768px;
```
Note: CSS custom properties cannot be used in @media queries directly. These are documented reference values. The actual media queries use the literal values.

**2. Add responsive base rules to css/base.css** (after component definitions):

```css
/* ── Responsive: Small phone floor (<360px) ── */
@media (max-width: 359px) {
  :root {
    --sp-2: 12px;
    --sp-3: 16px;
  }
}

/* ── Responsive: Tablet (>=768px) ── */
@media (min-width: 768px) {
  .screen {
    max-width: 640px;
    margin-inline: auto;
  }
}
```

The small phone override reduces spacing to prevent overflow. The tablet rule centers content with a max-width so screens don't stretch full-width on tablets.

**3. Fix streak dot touch targets in css/dashboard.css:**

Find `.streak-dot` (currently at line 103):
```css
.streak-dot {
  width: 28px;   /* CHANGE to: min-width: 44px */
  height: 28px;  /* CHANGE to: min-height: 44px */
  ...
}
```

Replace with:
```css
.streak-dot {
  min-width: 44px;
  min-height: 44px;
  border-radius: 8px;
  border: 1.5px solid var(--c-border);
  display: flex;
  align-items: center;
  justify-content: center;
  transition: all var(--t-fast);
}
```

The key change: `width: 28px; height: 28px;` becomes `min-width: 44px; min-height: 44px;`. This ensures the touch target meets the 44px WCAG requirement. The inner SVG icon stays at 12px (line 120-122, no change needed), so the visual remains proportional with more padding around the icon.

**4. Adjust streak grid if needed for 44px dots:**
Check the parent container of streak dots (likely `.streak-grid` or similar) — if it uses a fixed grid with 28px assumption, update the column size to accommodate 44px cells. The grid should use `repeat(7, 1fr)` or `repeat(7, 44px)` to fit 7 days. If the container is narrower than 7*44px=308px, the grid will shrink via minmax or fr — this works fine even on 360px screens (360 - 2*16px padding = 328px > 308px).
